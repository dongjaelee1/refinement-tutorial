# refinement-tutorial
Tutorial for refinement based verification

Work In Progress.

## Build
Requirement: opam (>=2.0.0), Coq 8.15.0
- Install dependencies with opam
```
./configure
```
- Build the project
```
make build -j
```

## Structure
All the files are in `src/tutorial/`.
1. `Refinement.v`
2. `Imp.v`
3. `FiniteSimulation.v`
4. `Example1.v`
6. `Simulation.v`
7. `Example2.v`
